* Technologies:
   * _Java 8_
   * _Log4j2_
   * _Spring boot 2_
   * _Spring 5_
   * _Hibernate 5_
   * _MySQL 8_
* Unit and Integration application tests
* Using _GitHub Actions_ to Build a Project
* Using _Liquibase_ for Migration
